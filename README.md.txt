#Task Manager REST API

This is a simple RESTful API for managing tasks, 
built using **ASP.NET Core 8** and **Entity Framework Core** with an in-memory database.

---

##Features

- Create, Read, Update, Delete tasks
- REST API with JSON input/output
- Follows RESTful conventions
- Uses in-memory DB (no external setup required)
- Swagger UI for easy testing
